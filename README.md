# Elite Digital Hub Store

متجر عربي سريع ومصمم بأسلوب Mobile‑First لبيع اشتراكات الذكاء الاصطناعي والخدمات الرقمية، مع سلة طلب وإتمام الطلب عبر واتساب.

## التشغيل محليًا

```bash
npm start
```

ثم افتح `http://localhost:4173`.

## المميزات

- تصميم متجاوب يبدأ من الموبايل.
- 11 عرضًا مرتبًا بوضوح مع طلب مباشر لكل منتج.
- كل زر منتج يفتح رقم واتساب مباشرة برسالة طلب جاهزة.
- CTA واتساب واضح داخل الـHero Section.
- بيانات SEO وStructured Data.
- لا توجد مكتبات تشغيل أو خطوط خارجية، لتحميل أسرع.

## التخصيص

- المنتجات والأسعار: `index.html`
- رقم واتساب: `script.js` وروابط واتساب في `index.html`
- الألوان والتصميم: المتغيرات أعلى `styles.css`

قبل النشر على دومين نهائي، أضف رابط الموقع في `canonical` وبيانات Open Graph، ثم أنشئ `sitemap.xml` بالدومين الحقيقي.
